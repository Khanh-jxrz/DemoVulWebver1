Một trang web bán hàng chứa các lỗ hổng bảo mật Web

Các lỗ hổng :XSS, SQLi , FileUpload , SSRF , IDOR.

Hướng dẫn cài :

```
git clone https://github.com/Khanh-jxrz/DemoVulWebver1.git
```
```
cd DemoVulWebver1 
```
```
sudo docker-compose up --build -d
```

Truy cập vào địa chỉ http://localhost:8080
